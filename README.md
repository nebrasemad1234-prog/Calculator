# آلة حاسبة بسيطة مع اختبارات Unit Test

## وصف المشروع

هذا المشروع عبارة عن آلة حاسبة بسيطة مكتوبة بلغة Python تقوم بتنفيذ العمليات الحسابية الأساسية:

- الجمع
- الطرح
- الضرب
- القسمة

كما يحتوي المشروع على اختبارات Unit Test للتأكد من أن كل دالة تعمل بشكل صحيح.

---

## هيكل المشروع

```

project-folder/
│
├── calculator.py        # دوال الآلة الحاسبة
├── test_calculator.py   # اختبارات Unit Test
└── README.md            # وصف المشروع

```

---

## المتطلبات

- Python 3 أو أحدث

لا يحتاج المشروع إلى أي مكتبات خارجية، حيث يستخدم مكتبة `unittest` المدمجة في بايثون.

---

## طريقة تشغيل البرنامج

لتشغيل الآلة الحاسبة:

```

python calculator.py

```

بعد التشغيل سيطلب منك:
1. اختيار العملية
2. إدخال الرقم الأول
3. إدخال الرقم الثاني

وسيظهر الناتج.

---

## طريقة تشغيل الاختبارات

لتشغيل اختبارات Unit Test:

```

python -m unittest test_calculator.py

```

أو:

```

python test_calculator.py

```

إذا نجحت جميع الاختبارات ستظهر رسالة تفيد بذلك.

---

## ما الذي يتم اختباره؟

يقوم البرنامج باختبار:

- صحة عملية الجمع
- صحة عملية الطرح
- صحة عملية الضرب
- صحة عملية القسمة
- التأكد من ظهور خطأ عند القسمة على صفر
